# SNISID Root CA Key Ceremony Playbook
**Classification:** RESTRICTED
**Standard Compliance:** ANSSI RGS / NIST SP 800-57

## 1. Prerequisites & Physical Controls
- **Location:** Faraday cage / Secure room without network access (Air-gapped).
- **Hardware:**
  - 1x Dedicated, newly imaged laptop (no network interfaces enabled).
  - 1x Offline Hardware Security Module (HSM) (e.g., Utimaco/Thales).
  - 3x Encrypted USB drives for backups.
- **Personnel Quorum (M-of-N):** 3 out of 5 Key Custodians must be physically present.
- **Auditor:** 1x External Auditor (or DPO) to record the ceremony.

## 2. Environment Preparation
1. The Auditor verifies the checksum of the OS image on the dedicated laptop.
2. The laptop is booted from the sovereign ISO (`MASTER_BOOTABLE_ISO_PROMPT` output).
3. The physical HSM is connected via USB/PCIe.

## 3. Cryptographic Execution
1. **Initialize the HSM:**
   - Format the HSM.
   - Generate the Master Key using Shamir's Secret Sharing (M=3, N=5).
   - Distribute the 5 Smart Cards to the 5 Key Custodians.

2. **Generate the Root CA Keypair inside the HSM:**
   - `pkcs11-tool --module /usr/lib/libcs_pkcs11_R2.so --keypairgen --key-type rsa:4096 --label "SNISID_Root_CA"`
   - *Ensure the private key is marked as non-extractable.*

3. **Generate the Root CA Certificate:**
   - Run OpenSSL using the `openssl-root.cnf` file, passing the PKCS#11 engine to sign the certificate using the HSM.
   - Validity: 20 Years.
   - Export the public `snisid_root_ca.cert.pem`.

4. **Sign the Intermediate CA (HashiCorp Vault):**
   - Import the Certificate Signing Request (CSR) generated by the online HashiCorp Vault cluster via USB.
   - Sign the CSR using the Root CA, applying the `[ v3_intermediate_ca ]` profile to enforce `pathlen:0`.
   - Export the signed `vault_intermediate.cert.pem` to the USB.

## 4. Teardown & Secure Storage
1. Shut down the air-gapped laptop. Destroy the RAM (wait 5 minutes / cold boot attack prevention).
2. The physical HSM is placed in the Level 5 Security Safe.
3. The 5 Key Custodians return their Smart Cards to their personal biometric safes.
4. The Auditor signs the formal Key Ceremony Log.
