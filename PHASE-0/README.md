## 🎯 Objet de la Phase 0

La Phase 0 est la **fondation stratégique absolue** du programme SNISID. Elle établit l'ensemble du cadre :
- **Vision nationale** 2026–2030
- **Gouvernance** de l'infrastructure critique
- **Cadre légal** complet (7 lois + décrets)
- **Architecture souveraine** de référence
- **Modèle de données** national
- **Stratégie offline-first** haïtienne
- **Cadre de cybersécurité** national
- **Interopérabilité** inter-agences
- **Workflows** industrialisés
- **Stratégie datacenter** souverain
- **Opérations 24/7**
- **Modèle financier** 2026–2030
- **Registre des risques** national
- **Roadmap** maîtresse 2026–2030

> **RÈGLE D'OR :** Aucune ligne de code ne sera écrite avant validation complète de ce repository par l'Autorité Nationale Numérique.

## 🔐 Principes Zero Trust

```
NEVER TRUST — ALWAYS VERIFY — ASSUME BREACH

1. Vérification identité : FIDO2/WebAuthn + Keycloak (OIDC)
2. Autorisation contextuelle : OPA Rego (ABAC)
3. Chiffrement ubiquitaire : mTLS via Istio + TLS 1.3
4. Microsegmentation : Cilium eBPF NetworkPolicies
5. Secrets dynamiques : HashiCorp Vault (TTL courts)
6. Audit immuable : WORM storage + event-sourcing
7. Vérification continue : tokens courts (15 min max)
```
