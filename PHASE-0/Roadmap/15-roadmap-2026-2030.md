## 4. Phase 2 — BUILD (2027 S2 – 2028)

### Objectifs
- Construire la plateforme technique core
- Mettre en service le datacenter PaP
- Démarrer le datacenter DR Cap-Haïtien
- Lancer pilote dans 2 départements

### Jalons

| Trimestre | Jalon |
|-----------|-------|
| 2027 Q3 | Plateforme K8s + Kafka + Keycloak opérationnelle (env. dev/test) |
| 2027 Q3 | PKI nationale opérationnelle (cérémonie clés) |
| 2027 Q4 | Datacenter PaP mis en service |
| 2027 Q4 | Migration données ONI legacy commencée |
| 2027 Q4 | Module état civil pilote (Naissance Simple) en recette |
| 2028 Q1 | Pilote terrain — Département Ouest (PaP + banlieue) |
| 2028 Q2 | 5 agences connectées au bus national (ONI, MJSP, CNIGS, PNH, DIE) |
| 2028 Q2 | AFIS national 1:N en production |
| 2028 Q3 | Datacenter DR Cap-Haïtien opérationnel |
| 2028 Q3 | Pilote étendu — Département Nord |
| 2028 Q4 | Bilan pilote + recette finale |
| 2028 Q4 | SOC national opérationnel 24/7 |

### Livrables fonctionnels Phase 2
- Enrôlement CIN (online + offline)
- Workflows naissance (5 types)
- Workflows mariage, divorce, décès
- Délivrance acte numérique signé
- Vérification 1:1 et 1:N
- Bus national + 5 APIs publiées par agence pilote

## 5. Phase 3 — DEPLOYMENT (2029)

### Objectifs
- Déploiement national des 10 départements
- Campagne nationale d'enrôlement
- Onboarding de 5 agences supplémentaires
- Stabilisation et industrialisation

### Jalons

| Trimestre | Jalon |
|-----------|-------|

| 2029 Q1 | 50+ Offline kits déployés |

| 2029 Q2 | 3M citoyens enrôlés |
| 2029 Q3 | Service KYC pour banques en production |
| 2029 Q3 | Liste électorale automatiquement alimentée (CEP) |
| 2029 Q4 | 5M citoyens enrôlés |
| 2029 Q4 | Numérisation state civil historique : ≥ 50 % |

## 6. Phase 4 — SOUVERAINETÉ (2030)

### Objectifs
- Couverture 95 % de la population
- 15 agences interconnectées
- Trajectoire de soutenabilité financière confirmée
- Autonomie opérationnelle complète
- Reconnaissance internationale

### Jalons

| Trimestre | Jalon |
|-----------|-------|
| 2030 Q1 | 8M citoyens enrôlés (~75 %) |
| 2030 Q1 | Onboarding INARA, ONA, CEP, BRH, CONATEL |
| 2030 Q2 | Service signature électronique qualifiée public ouvert |
| 2030 Q3 | 10M+ enrôlés (≥ 95 %) |
| 2030 Q3 | Audit ISO 27001 obtenu |
| 2030 Q4 | Bilan ODD 16.9 — Identité légale pour tous |
| 2030 Q4 | Rapport au Parlement — Phase 5 (2031-2035) lancée |
