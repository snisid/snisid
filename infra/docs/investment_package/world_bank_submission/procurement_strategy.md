# SNISID: Project Procurement Strategy for Development (PPSD)
## Tendering Framework for Sovereign Infrastructure

---

### 1. Procurement Principles
- **Transparency:** All tenders published on the SNISID portal and WB UN Development Business (UNDB).
- **Open Competition:** Prioritizing International Competitive Bidding (ICB) for high-value contracts.
- **Value for Money:** Evaluation based on a weighted 70/30 Technical/Financial score.
- **Auditability:** Every step of the procurement process tracked in the World Bank STEP system.

### 2. Major Procurement Packages

#### Package 1: National Data Center & Cloud Hardware ($20M)
- **Method:** International Competitive Bidding (ICB).
- **Scope:** Servers, HSMs, storage, and networking for the primary and DR sites.

#### Package 2: Identity & SOC Software Core ($15M)
- **Method:** Quality and Cost Based Selection (QCBS).
- **Scope:** Implementation of the Go/gRPC backend, Kafka streaming, and AI fraud models.

#### Package 3: Biometric Enrollment & Field Kits ($10M)
- **Method:** National Competitive Bidding (NCB) / International Competitive Bidding (ICB).
- **Scope:** Ruggedized mobile enrollment units and biometric scanners.

#### Package 4: Technical Assistance & Governance Consulting ($5M)
- **Method:** Quality Based Selection (QBS).
- **Scope:** Legal drafting, security auditing, and institutional capacity building.

### 3. Anti-Corruption Safeguards
- **Mandatory Disclosure:** All bidders must disclose beneficial ownership.
- **Conflict of Interest:** Strict exclusion rules for evaluators with agency ties.
- **Grievance Redress Mechanism:** A formal portal for bidders to contest results.

### 4. Selection Methods Summary
| Item Type | Primary Method |
| :--- | :--- |
| **Complex Software/Systems** | QCBS (70/30) |
| **Standard Hardware** | Lowest Evaluated Substantially Responsive Bid (LEB) |
| **Highly Specialized Consulting** | QBS |
| **Small Value Goods/Works** | Shopping (National) |
