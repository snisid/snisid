🧠 0) Hypothèses & principes
SNISID = plateforme distribuée (Windows en edge + Linux/K8s au cœur).
Multi-agences (ANH / DGI / DCPJ / ONI) avec RBAC + mTLS.
Active/Active entre 2 sites principaux + 1 site de reprise (DR).
Kafka comme backbone, Neo4j pour graph identité, GPU pour IA.
RPO ≤ 5 min, RTO ≤ 30 min (cible réaliste).
🧱 1) PLAN CLUSTER NATIONAL (10 SERVEURS)

🧠 2) STRUCTURE LOGICIELLE (rappel aligné infra)
Core: API Gateway (Go), SOC Engine, Policy (OPA)
Streaming: Kafka + Flink (topics: photo.ingested → verified → fraud.detected)
Graph: Neo4j (identités, devices, liens inter-agences)
AI: ArcFace, Deepfake (GPU)
Realtime: WebSocket Gateway → UI
Stockage: MinIO (S3) + snapshots Neo4j + Kafka logs
🧮 3) CAPACITÉ & PERFORMANCE CIBLES
Ingestion: 100k–500k events/sec (cluster)
Latence alerte SOC: 50–150 ms
Reconnaissance visage: 20–80 ms / image (GPU)
Deepfake: 100–300 ms / média (selon modèle)
Graph queries: < 200 ms (indexes + cache)
💾 4) STORAGE & RÉTENTION
Hot (NVMe): 7–30 jours (Kafka + Neo4j actifs)
Warm (HDD RAID10 / MinIO): 3–6 mois
Cold (object storage + backup offsite): 1–5 ans

Règles

Kafka: compaction + retention (ex: 7–14j)
Neo4j: snapshots journaliers + WAL
MinIO: versioning + lifecycle
🔐 5) SÉCURITÉ INTER-AGENCES
🔑 Identité & accès

mTLS (Istio) → chiffrement east-west
JWT + RBAC (ANH/DGI/DCPJ/ONI)
🧾 RBAC (exemple)

🛡️ OPA (policy)
filtrage par rôle + audit obligatoire

💰 8) COÛT ESTIMATIF (ordre de grandeur)
🖥️ Hardware (10 serveurs)
Serveur standard (sans GPU): $6k–10k × 8 → $48k–80k
Serveur GPU (2×3090): $12k–18k × 2 → $24k–36k
NIC 25–100GbE + switch: $20k–60k
Disques (NVMe + HDD): $20k–40k

👉 Total hardware: ~$110k → $200k

🏢 Infra (DC + réseau + énergie)
racks, UPS, refroidissement, câblage: $30k–80k
🔧 Opex (annuel)
électricité, internet, maintenance: $20k–60k
