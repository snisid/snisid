🧱 5. ENFORCEMENT POINT (CRITICAL)

Every microservice must call:

Policy Decision Point (PDP)

Go enforcement middleware:
func PolicyMiddleware(next http.Handler) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {

        input := BuildPolicyInput(r)

        allowed := CallPolicyEngine(input)

        if !allowed {
            http.Error(w, "POLICY_DENIED", 403)
            return
        }

        next.ServeHTTP(w, r)
    })
}
