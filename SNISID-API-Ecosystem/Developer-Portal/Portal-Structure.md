# SNISID National Developer Portal

## Purpose
A single entry point for government developers to discover, test, and integrate with national APIs.

## Key Features
- **API Documentation**: Interactive Swagger/Redoc documentation.
- **SDK Downloads**: Generated clients for Java, Python, Go, and Node.js.
- **API Sandbox**: Mock environment for testing without real data.
- **Key Management**: Self-service API key and credential rotation.
- **Tutorials & Guides**: Onboarding documents for new agency integrations.

## Governance Integration
- Integration requests must go through an approval workflow within the portal.
- Monthly usage reports available for agency admins.
