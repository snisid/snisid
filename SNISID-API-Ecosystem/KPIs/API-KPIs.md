# SNISID National API KPIs

## Primary Performance Indicators

| KPI | Description | Target |
| :--- | :--- | :--- |
| **Availability (Uptime)** | Percentage of time the Gateway is operational. | > 99.95% |
| **Mean Time To Recovery (MTTR)** | Average time to fix an API failure. | < 30 mins |
| **Error Rate** | Ratio of 5xx errors to total requests. | < 0.1% |
| **P95 Latency** | Time for 95% of requests to complete. | < 200ms |
| **API Adoption** | Number of agencies integrated. | 100% by Y2 |
| **Security Violations** | Number of unauthorized access attempts. | Goal: 0 |
| **Documentation Quality** | Developer satisfaction survey. | > 4.5/5 |

## Monitoring Frequency
- **Real-time**: Uptime, Error Rate, Latency.
- **Monthly**: Adoption, Security Violations, MTTR.
- **Quarterly**: Developer satisfaction.
