# SNISID Offline-First National Resilience Repository

This directory contains the complete offline-first architecture, engines, workflows, and runbooks for the Haitian National Identity System (SNISID).

## Structure
- **Edge-Nodes/**: Regional departmental autonomy architecture
- **Synchronization/**: Delayed sync engine specifications
- **Conflict-Resolution/**: Automated + human conflict handling
- **Mobile-Units/**: Field enrollment and verification units
- **Offline-IAM/**: Offline identity verification and security
- **Offline-BPMN/**: Offline-adapted workflows
- **Energy/**: Power resilience strategy
- **Crisis-Mode/**: National emergency operations
- **Observability/**: Monitoring during outages
- **Runbooks/**: Operational procedures for all failure scenarios
