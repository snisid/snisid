# Runbook: Internet Outage Continuity

## Immediate Actions
1. Activate local edge autonomy
2. Switch to offline mode for all workflows
3. Enable local biometric verification only
4. Queue all events with timestamps

## Recovery
1. Monitor connectivity
2. Initiate prioritized sync (emergency first)
3. Run conflict resolution
4. Reconcile national registry

## Escalation
- After 72h: Activate regional crisis protocols
- After 7 days: National emergency mode
