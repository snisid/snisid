# Runbook: Power Failure Resilience

## Edge Node Response
1. Automatic switch to battery/solar
2. Enter low-power mode (reduce non-critical services)
3. Preserve identity cache in non-volatile storage

## Mobile Units
1. Deploy solar panels immediately
2. Prioritize biometric devices and laptops

## Core
1. Activate UPS and generators
2. Notify regional nodes of degraded mode
