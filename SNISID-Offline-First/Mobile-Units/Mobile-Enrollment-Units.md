# Mobile Enrollment Units

## Hardware Components
- Rugged laptops/tablets
- Portable biometric devices (fingerprint, iris, facial)
- Solar power kits + high-capacity batteries
- Satellite communication terminal (Iridium/Starlink backup)
- Local offline node (edge computing device)

## Operational Capabilities
- Full enrollment in zero-infrastructure zones
- Local identity verification
- Event buffering for later sync
- GPS-tagged operations with offline maps

## Deployment Scenarios
- Remote rural areas
- Disaster zones
- Border and migration points
