# Offline Observability Model

## Metrics Collected Offline
- Sync lag (when reconnected)
- Offline duration per node
- Local queue depth
- Conflict rate
- Energy levels
- Biometric match success rate

## Tools
- Local Prometheus + Grafana instances (edge)
- Event logs with cryptographic signing
- Post-sync reconciliation dashboards

## National Dashboard (upon reconnection)
- Heatmap of offline regions
- Critical queue backlog alerts
- Conflict resolution status
