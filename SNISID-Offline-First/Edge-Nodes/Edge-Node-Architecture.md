# Regional Edge Nodes Architecture

## Requirements per Department
- Kubernetes edge node for local runtime
- Local cache for identity data
- Local IAM (Identity and Access Management)
- Local BPMN workflow execution
- Local synchronization queue

## Autonomy Features
- Operate independently for 7-30 days
- Local biometric verification
- Offline identity issuance (provisional)
- Local judicial status checks (cached)

## Deployment
- 10 departments + Port-au-Prince special zone
- Solar + battery backup
- Satellite fallback for priority sync
