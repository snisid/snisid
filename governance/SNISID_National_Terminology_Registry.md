# SNISID National Terminology Registry
**Version** : 1.0 | **Date** : 2025-03-27 | **Statut** : ✅ Approuvé

| Terme               | Définition                                                                 |
|---------------------|-----------------------------------------------------------------------------|
| NNU                 | Numéro National Unique, identifiant unique d’une personne physique.         |
| Enrollment          | Processus de création d’une identité dans SNISID (biométrie + documents).   |
| Verification        | Validation d’une identité existante (1:1).                                  |
| Identification      | Recherche d’une personne dans la base (1:N).                                |
| Identity Recovery   | Processus de réémission d’identité après perte ou vol.                      |
| Revocation          | Invalidation définitive d’une identité (ex : décès, naturalisation).        |
| Judicial Flag       | Marque judiciaire attachée à une identité (mandat d’arrêt, interdiction).   |
| Edge Node           | Serveur local d’une commune, fonctionnant en mode dégradé.                  |
| Sync Batch          | Paquet d’événements transmis en masse lors de la synchronisation.           |
| Attestation         | Document numérique (PDF signé) prouvant une identité.                       |
| Token               | Jeton d’accès OAuth2 ou JWT temporaire.                                     |
| CA                  | Autorité de Certification (Root CA nationale).                              |
| Certificate         | Certificat X.509 émis par la CA nationale.                                  |
