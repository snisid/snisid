# SNISID — CADRE D'ALIGNEMENT CONSTITUTIONNEL

**Classification :** CADRE JURIDIQUE FONDAMENTAL — CONSTITUTIONNEL
**Référence :** SNISID-CONST-001
**Version :** 1.0
**Date :** 25 mai 2026

---

## 1. OBJECTIF

Garantir que le SNISID respecte pleinement la Constitution de la République d'Haïti de 1987 (amendée), ses principes fondamentaux, et l'ensemble des droits et libertés qu'elle consacre.

---

## 2. PRINCIPES CONSTITUTIONNELS APPLICABLES

La Constitution haïtienne consacre des droits fondamentaux que le SNISID doit impérativement respecter. Ce cadre établit la correspondance entre les principes constitutionnels et les garanties techniques et organisationnelles du système.

---

## 3. GARANTIES CONSTITUTIONNELLES

### 3.1 Citizen Rights (Droits des Citoyens)

| Droit Constitutionnel | Article Constitution | Garantie SNISID |
|----------------------|---------------------|-----------------|
| Droit à la nationalité | Art. 11 | Le SNISID ne peut ni refuser ni retirer la nationalité — il enregistre les décisions légales |
| Droit à la personnalité juridique | Art. 16-17 | Tout citoyen a droit à un NNI et une identité numérique |
| Liberté individuelle | Art. 24 | Le SNISID ne restreint aucune liberté — il identifie, n'autorise ni n'interdit |
| Inviolabilité du domicile | Art. 44 | Les données de localisation ne sont pas utilisées pour la surveillance domiciliaire |
| Liberté d'expression | Art. 28 | Le SNISID ne surveille pas les opinions |
| Liberté de religion | Art. 30 | Aucune donnée religieuse collectée |
| Liberté d'association | Art. 31 | Aucun croisement données associatives/identité |
| Droit de propriété | Art. 36 | Les données personnelles sont protégées comme attribut de la personne |
| Égalité devant la loi | Art. 18 | Accès égal aux services SNISID sans discrimination |
| Droit à l'éducation | Art. 32 | NNI facilite l'accès à l'éducation, ne le restreint pas |
| Droit à la santé | Art. 19 | NNI facilite l'accès à la santé, ne le restreint pas |

**Garanties techniques :**
- Aucun champ de données ne capture l'opinion, la religion, l'affiliation politique
- Aucun algorithme de scoring social n'est autorisé
- L'identité numérique est un droit, pas un privilège
- Un citoyen ne peut être privé de services essentiels pour défaut d'enregistrement pendant la période de transition

### 3.2 Due Process (Procédure Régulière)

| Droit | Application SNISID |
|-------|-------------------|
| Droit d'être informé | Tout citoyen est informé des données collectées et de leur usage |
| Droit d'être entendu | Toute contestation d'une décision basée sur le SNISID peut être entendue |
| Droit de contestation | Toute décision automatisée peut être contestée devant un humain |
| Droit de recours | Recours administratif puis judiciaire disponible |
| Délai raisonnable | Traitement des demandes dans des délais définis |
| Motivation des décisions | Toute décision défavorable est motivée et documentée |

**Processus de contestation :**
```
Citoyen conteste une décision ou une donnée
    → Niveau 1 : Réclamation administrative (agence concernée) — 15 jours
    → Niveau 2 : Recours devant l'ANPD — 30 jours
    → Niveau 3 : Recours judiciaire (Tribunal administratif) — selon procédure
    → Niveau 4 : Appel — selon procédure judiciaire
    
À chaque niveau :
- Le citoyen est entendu
- Il peut être assisté d'un avocat
- La décision est motivée par écrit
- Les voies de recours sont indiquées
```

### 3.3 Judicial Oversight (Contrôle Judiciaire)

| Mécanisme | Description |
|-----------|-------------|
| Autorisation judiciaire préalable | Requise pour tout accès étendu aux données dans le cadre d'enquêtes |
| Mandat pour la collecte | Mandat judiciaire pour collecte de preuves numériques (sauf flagrant délit) |
| Contrôle de proportionnalité | Le juge vérifie la proportionnalité de l'accès aux données |
| Recours contre les décisions | Tout acte du SNISID affectant les droits peut être contesté devant le juge |
| Habeas data | Droit fondamental d'accès et de rectification de ses données, garanti judiciairement |
| Juge des libertés | Supervision de l'utilisation des données dans les procédures pénales |

**Garanties de contrôle judiciaire :**

| Situation | Autorisation Requise | Contrôle |
|-----------|---------------------|----------|
| Consultation standard (agent habilité) | Habilitation administrative | Journalisation + audit |
| Consultation étendue (enquête) | Réquisition judiciaire | Juge d'instruction |
| Accès aux données biométriques (enquête) | Mandat spécifique | Juge des libertés |
| Croisement de bases de données | Ordonnance judiciaire | Juge + ANPD |
| Surveillance en temps réel | Ordonnance motivée, durée limitée | Juge des libertés |
| Géolocalisation | Mandat spécifique, durée limitée | Juge des libertés |

### 3.4 Privacy Rights (Droits à la Vie Privée)

| Droit | Fondement | Garantie SNISID |
|-------|-----------|-----------------|
| Vie privée | Art. 44 Constitution | Privacy by Design, minimisation des données |
| Secret de la correspondance | Art. 47 | Aucune interception des communications citoyennes |
| Inviolabilité du domicile | Art. 44 | Pas de géolocalisation sans mandat judiciaire |
| Image | Droits fondamentaux | Photo utilisée uniquement pour identification, pas de surveillance |
| Données personnelles | Loi protection données | Consentement, accès, rectification, opposition |

**Privacy Impact Assessment constitutionnel :**
Avant tout nouveau traitement de données, une évaluation de conformité constitutionnelle est réalisée :

| Question | Réponse exigée |
|----------|---------------|
| Le traitement respecte-t-il les droits fondamentaux ? | Oui (documenté) |
| Le traitement est-il proportionné ? | Oui (justifié) |
| Des alternatives moins intrusives existent-elles ? | Analysé |
| Le citoyen peut-il s'opposer ? | Oui (sauf obligation légale) |
| Un recours est-il possible ? | Oui (administratif + judiciaire) |

### 3.5 Administrative Legality (Légalité Administrative)

| Principe | Application |
|----------|------------|
| Légalité | Tout acte du SNISID a une base légale |
| Compétence | Seules les autorités compétentes agissent |
| Motivation | Tout acte affectant les droits est motivé |
| Publicité | Les règles sont publiées et accessibles |
| Non-rétroactivité | Les nouvelles règles ne s'appliquent pas rétroactivement |
| Proportionnalité | Les mesures sont proportionnées à l'objectif |
| Recours | Tout acte est susceptible de recours |
| Responsabilité | L'État est responsable des dommages causés par le SNISID |

**Régime de responsabilité de l'État :**

| Situation | Responsabilité | Réparation |
|-----------|---------------|-----------|
| Erreur d'identité | État (ONI) | Correction + indemnisation |
| Fuite de données | État (agence responsable) | Notification + indemnisation |
| Décision automatisée erronée | État | Annulation + indemnisation |
| Accès illégal par agent | Agent + État | Sanctions + indemnisation |
| Dysfonctionnement système | État | Correction + indemnisation |
| Usage abusif des données | Agent + État | Sanctions pénales + indemnisation |

| Situation | Processus |
|-----------|----------|

## 5. LIMITES CONSTITUTIONNELLES DU SNISID

Le SNISID ne peut en aucun cas :

| Interdiction | Fondement |
|-------------|-----------|
| Créer un système de surveillance de masse | Libertés individuelles |
| Discriminer dans l'accès aux services | Égalité |
| Priver un citoyen d'identité sans décision judiciaire | Personnalité juridique |
| Partager des données sans base légale | Vie privée |
| Prendre des décisions automatisées sans recours | Due process |
| Restreindre la liberté de mouvement par le traçage | Liberté de circulation |
| Collecter des données sur les opinions/religion/politique | Libertés fondamentales |
| Refuser un service public essentiel pour défaut d'enregistrement | Droits sociaux |
